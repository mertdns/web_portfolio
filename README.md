# 🚀 Portfolio - Mert Danış

Fiverr platformunda freelance developer olarak hizmet vermek için tasarlanmış modern, koyu temalı ve gradient renkler içeren kişisel portfolio sitesi.

## ✨ Özellikler

- **Koyu Tema** - Göz yormayan, modern dark mode tasarımı
- **Gradient Renkler** - Mor, mavi ve pembe tonlarında etkileyici gradientler
- **Responsive Tasarım** - Mobil, tablet ve masaüstü cihazlarda mükemmel görünüm
- **Yüksek Performans** - Hafif animasyonlar, hızlı yüklenme
- **SEO Dostu** - Meta taglar ve semantic HTML yapısı

## 📁 Dosya Yapısı

```
portfolio/
├── index.html      # Ana HTML dosyası
├── styles.css      # CSS stilleri
├── script.js       # JavaScript fonksiyonları
└── README.md       # Bu dosya
```
